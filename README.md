# StoryOS Frontend

React frontend for StoryOS - Content management system for enterprise storytelling.

## Features
- UNF Elements View - Browse reusable content
- Deliverables View - Check for impact alerts  
- Real-time version tracking
- Clean Tailwind UI

## Quick Start
\`\`\`bash
npm install
npm run dev
\`\`\`
